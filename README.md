# SI101-Accumulators

## Getting Started

1. Fork this repository
2. Click **Code → Codespaces → Create Codespace on main**
3. Wait for the environment to finish setting up (1–2 minutes)
4. Open `notebook.ipynb`
5. When prompted, select the **Python 3.11** kernel

If imports fail:
- Wait another minute
- Reload the page
- Reopen the notebook
